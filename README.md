# P2kbot 
### Install
```
git clone https://github.com/e155/p2kbot/
export TELE_TOKEN="your token"
```
or
``` echo export TELE_TOKEN="your token" >> ~./bashrc ```
### Run
```./p2kbot kbot|start```

Call bot: https://t.me/e155_p2kbot
###  Using bot commands:
/start hello - to say Hello

/start version - to tell version
